#    Install   #

1 - Open a terminal (Ctrl+Alt+T) and type:
	sudo add-apt-repository ppa:swi-prolog/stable

2 - Afterwards, update the package:
	sudo apt-get update

3 - Install SWI-Prolog:
	sudo apt-get install swi-prolog


#    Run  #

1 - Write a prolog program. For example, program.pl.
2 - Open a terminal (Ctrl+Alt+T) and navigate to the directory where you stored your program.
3 - Now type swipl -s program.pl.
4 - In order to query the loaded program, type goals and watch the output. For exemple, " fatorial(0, X). "
